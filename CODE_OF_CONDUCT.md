# Code of Conduct

This project adopts the **Contributor Covenant v2.1** as its community standard.

The full text and enforcement guidelines are available at the canonical source:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Conduct concerns can be reported to the repository maintainer via GitHub (open a private security advisory) or by email to the maintainer listed in `Cargo.toml`.

Reports are handled in confidence.
